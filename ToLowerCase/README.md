# To Lower Case

Given a string s, return the string after replacing every uppercase letter with the same lowercase letter.

### Example1
```sh
Input: s = "Hello"
Output: "hello
```

### Example2
```sh
Input: s = "here"
Output: "here"
```

### Example3
```sh
Input: s = "LOVELY"
Output: "lovely"
```

### Constraints:
```sh
1 <= s.length <= 100
s consists of printable ASCII characters.
```
