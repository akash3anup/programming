# Nth Fibonacci Number Modulo M

Given two integers n and m, output F n mod m (that is, the remainder of F n when divided by m).

### Example 1
```sh
Input: n = 7, m = 3
Output: 1
```

### Example 2
```sh
Input: n = 2015, m = 3
Output: 1
```

### Example 3
```sh
Input: n = 239, m = 1000
Output: 161
```

### Example 4
```sh
Input: n = 2816213588, m = 239
Output: 151
```

### Constraints
```sh
1 <= n <= 10^14
2 <= m <= 10^3
```
